viewports

viewports is a Svelte Store, which returns true whenever a Viewports selected in initialization is currently active.

NOTE: The REPL currently does not support viewport values. Resize your Browser instead.

Imports

import {viewports} from "@kahi-ui/framework";

Compatibility

The Store is dependent on Window.matchMedia. So if you're rendering on the server, it'll always return false.
